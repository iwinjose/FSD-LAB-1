# Turf Arena | Premium Online Turf Booking

Turf Arena is a modern, responsive web application for booking sports turfs online. It provides users with an intuitive interface to browse available turfs, check slot availability, and make instant bookings for football, cricket, and other sports facilities.

## Features

- **Modern UI/UX**: Premium design aesthetics utilizing Tailwind CSS, glassmorphism effects, and smooth micro-animations.
- **User Authentication**: Login and Registration pages for secure user access.
- **Slot Booking**: Browse and select available time slots.
- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Styling**: Tailwind CSS (via CDN)
- **Fonts**: Google Fonts (Inter, Outfit)

## Getting Started

To view and interact with the project locally:

1. Clone the repository or download the source code.
2. Open `index.html` in any modern web browser.
3. Navigate through the application to explore the features, login, and registration pages.

## Project Structure

- `index.html`: Main landing and booking page.
- `login.html`: User login interface.
- `register.html`: User registration interface.
- `css/style.css`: Custom fallback styles and overrides.
- `js/script.js`: Interactive logic and dynamic functionality.
- `images/`: Directory containing project assets.
